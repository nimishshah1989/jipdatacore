Intelligence Layer

Structured intelligence
from unstructured content

Market research, analyst commentary, and investment ideas are automatically ingested, structured, and made searchable — eliminating the manual effort of reading and tagging documents.

📰
Goldilocks Research
Daily Trend Friend reports automatically ingested from Goldilocks Research. Each report is structured into market views, sector calls, and stock ideas.
💡
Stock Ideas Tracking
Every stock recommendation is captured with entry price, target, stop-loss, and rationale. Performance tracked automatically against live price data.
🎙️
Audio & Video Transcription
Interviews, podcasts, and webinars are transcribed and structured. Key themes, mentioned stocks, and investment views are extracted and stored as searchable data.
🤖
AI-Powered Extraction
Claude AI reads each document and extracts structured investment views — bullish/bearish stance, time horizon, conviction level — in a consistent, comparable format.
Technology

Built for reliability,
not complexity

We deliberately chose proven, boring infrastructure over trendy complexity. Every component is battle-tested, auditable, and runs without manual intervention.

🗄️
PostgreSQL on AWS
Single source of truth. Every platform reads from one verified database. No data silos, no inconsistencies.
⚡
FastAPI Service
High-performance API layer on AWS EC2. All platforms (MarketPulse, future apps) connect through this single gateway.
🔄
15+ Automated Pipelines
Every data source has a dedicated pipeline. They run on schedule, detect failures, and quarantine bad data automatically.
🔒
Encrypted PII Storage
Client-identifying data is encrypted with AES-256-GCM before storage. Bank-grade security, compliant by design.
✅
1,720 Automated Tests
Every calculation, every API endpoint, every pipeline is continuously tested. Regressions are caught before they reach production.
📊
Live Monitoring Dashboard
Pipeline health, row counts, data freshness — all visible in real time. No blind spots in our data operations.
