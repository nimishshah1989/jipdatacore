Design Decision Timeline
NSE ETF source: yfinance .NS, not BHAV copy
de_instrument only contains equities. The BHAV master_refresh pipeline loads from EQUITY_L.csv which excludes ETFs. yfinance with .NS suffix is the correct source for NSE ETF OHLCV data.
ETF-C1 · 2026-04-10
Backfill: yfinance max period, not Stooq tars
Same ticker format works for US ETFs, covers all exchanges, and avoids SCP/parse complexity. yfinance max period provides sufficient historical coverage without additional infrastructure.
ETF-C2 · 2026-04-10
38 NSE ETFs deactivated (is_active = FALSE)
Smaller AMC ETF variants (ICICI, HDFC, Kotak, UTI series) do not exist on yfinance. Rather than failing silently, set is_active=FALSE so downstream pipelines skip them cleanly.
ETF-C5 · 2026-04-10
Observatory counts now API-driven, not hardcoded
Removed hardcoded "130 ETFs", "2,743 instruments" etc. from the Observatory frontend. Tree metadata now pulls live from the coverage API, eliminating dashboard-backend name drift.
ETF-C6 · 2026-04-10
etf_prices pipeline handles all exchanges in one code path
Maps NSE tickers to .NS for yfinance, maps back for DB storage. A single pipeline handles NSE India and Global ETFs without separate code paths, preventing Dual Code Path Divergence.
